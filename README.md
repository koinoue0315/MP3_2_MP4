# MP3 to MP4

## Features
音声ファイルと静止画1枚から動画を作成する

## Requirement
- python 3.x
- ffmpeg n4.4-19-g8d172d9409

## Installation

```bash
pip install mutagen
```

## Usage
input_audioフォルダにmp4化したいファイルを配置して下記コマンドを実行する


```
python script.py
```

生成されたmp4ファイルはoutputディレクトリに出力される
